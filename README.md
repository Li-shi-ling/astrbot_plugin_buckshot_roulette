# 恶魔轮盘 QQOfficial

AstrBot 插件：面向 QQ 官方机器人群聊的多人无庄家恶魔轮盘，支持 2-10 人房间、公开弹数、道具、回合推进、淘汰和胜负结算。

当前版本：`v1.2.7`

## 功能概览

- 支持 2-10 名玩家在 QQ 官方机器人群聊中进行同一局游戏。
- 支持创建、加入、退出、开始、状态查询和主动结束房间。
- 支持向自己或指定编号玩家开枪，并自动处理回合推进、淘汰和胜负结算。
- 支持啤酒、香烟、锯子、手铐、放大镜、过期药、电话和转变器八种道具。
- 支持子弹数量、道具刷新数量、道具持有上限和开局血量设置。
- 支持群内昵称持久化绑定，并提供名称绑定菜单按钮。
- 使用 QQ 官方 Markdown 指令按钮；客户端不显示按钮时仍可使用完整文本指令。

## 快速开始

在 QQ 群中发送：

```text
@机器人 轮盘菜单
```

首次使用时，请点击“名称绑定”，将自动生成的 `@机器人 名称绑定 [名称]` 中 `[名称]` 改成自己的名称后发送。随后可通过菜单创建房间、调整设置、查看帮助和查看道具效果。

## 命令

### 菜单与身份

| 命令 | 说明 | 示例 |
|---|---|---|
| `轮盘菜单` | 打开轮盘按钮菜单 | `@机器人 轮盘菜单` |
| `名称绑定 [名称]` | 绑定自己的群内游戏名称 | `@机器人 名称绑定 玩家一` |
| `轮盘绑定 [名称]` | 名称绑定的兼容指令 | `@机器人 轮盘绑定 玩家一` |
| `轮盘我的名字` | 查看当前绑定名称 | `@机器人 轮盘我的名字` |
| `轮盘改名 [新名称]` | 修改绑定名称 | `@机器人 轮盘改名 新名字` |
| `轮盘帮助` | 查看玩法与指令说明 | `@机器人 轮盘帮助` |
| `轮盘道具查看` | 查看全部道具效果 | `@机器人 轮盘道具查看` |

### 房间与对局

| 命令 | 说明 | 示例 |
|---|---|---|
| `轮盘创建` | 创建房间并成为房主 | `@机器人 轮盘创建` |
| `轮盘加入` | 加入当前群房间 | `@机器人 轮盘加入` |
| `轮盘退出` | 在开局前退出房间 | `@机器人 轮盘退出` |
| `轮盘开始` | 房主开始游戏，至少需要 2 人 | `@机器人 轮盘开始` |
| `轮盘状态` | 查看玩家血量、道具和当前行动者 | `@机器人 轮盘状态` |
| `轮盘开枪 自己` | 向自己开枪 | `@机器人 轮盘开枪 自己` |
| `轮盘开枪 [编号]` | 向指定编号玩家开枪 | `@机器人 轮盘开枪 2` |
| `轮盘结束` | 房主或管理员结束当前游戏 | `@机器人 轮盘结束` |

### 道具

| 命令 | 效果 |
|---|---|
| `轮盘道具 啤酒` | 移除当前第一发子弹，并公开它的类型 |
| `轮盘道具 香烟` | 恢复 1 点血量，但不能超过最大血量 |
| `轮盘道具 锯子` | 下一发实弹伤害增加 1；若为空弹则效果消失 |
| `轮盘道具 手铐 [编号]` | 使指定的其他玩家跳过下一次行动 |
| `轮盘道具 放大镜` | 查看当前第一发是实弹还是空弹 |
| `轮盘道具 过期药` | 随机恢复 2 点血量或失去 1 点血量 |
| `轮盘道具 电话` | 随机查看弹队列中某一发子弹的类型 |
| `轮盘道具 转变器` | 反转当前第一发子弹的类型 |

“转变器”同时兼容旧名称“转向器”“换向器”和“逆转器”。手铐是目标型道具；点击手铐按钮后会显示可选目标，也可以直接填写玩家编号。

### 游戏设置

发送 `@机器人 轮盘设置` 可查看当前公开默认设置。未创建房间时也可以修改，新房间会复制修改后的设置。

| 命令 | 说明 |
|---|---|
| `轮盘设置 子弹上限 [数量]` | 设置每轮子弹数量上限 |
| `轮盘设置 子弹下限 [数量]` | 设置随机子弹数量下限 |
| `轮盘设置 随机子弹 是/否` | 是否随机生成每轮子弹数量 |
| `轮盘设置 道具刷新上限 [数量]` | 设置每次装填的道具刷新上限 |
| `轮盘设置 道具刷新下限 [数量]` | 设置随机道具刷新下限 |
| `轮盘设置 随机道具 是/否` | 是否随机生成道具刷新数量 |
| `轮盘设置 道具持有上限 [数量]` | 设置每名玩家的道具持有上限 |
| `轮盘设置 血量上限 [数量]` | 设置固定血量或随机血量上限 |
| `轮盘设置 血量下限 [数量]` | 设置随机血量下限 |
| `轮盘设置 随机血量 是/否` | 是否在开局时随机血量 |

数量按钮会生成带 `[数量]` 占位符的指令，需要改成实际数字后发送。开启随机血量后，每局开始时只随机一次，并将相同初始血量应用于所有玩家。

## 游戏机制

- 开局会公开本轮实弹和空弹数量，但不会公开弹队列顺序。
- 向自己打出空弹时保留当前回合；其他开枪结果会推进回合。
- 弹队列耗尽后自动重新装填，并向所有存活玩家发放本轮道具。
- 淘汰玩家会从后续行动和道具发放中跳过。
- 场上只剩一名存活玩家时自动结束并宣布获胜者。

## 配置说明

配置文件为 `_conf_schema.json`，配置项均带有 AstrBot 配置界面提示文本。

| 配置项 | 说明 |
|---|---|
| `roulette_settings.shell_count_max` | 每轮子弹数量上限 |
| `roulette_settings.shell_count_min` | 随机子弹数量下限 |
| `roulette_settings.random_shell_count` | 是否随机生成每轮子弹数量 |
| `roulette_settings.item_count_max` | 每轮道具刷新上限 |
| `roulette_settings.item_count_min` | 随机道具刷新下限 |
| `roulette_settings.random_item_count` | 是否随机生成道具刷新数量 |
| `roulette_settings.item_inventory_max` | 每名玩家的道具持有上限 |
| `roulette_settings.hp_max` | 固定血量或随机血量上限 |
| `roulette_settings.hp_min` | 随机血量下限 |
| `roulette_settings.random_hp` | 是否在开局时随机一次全员血量 |
